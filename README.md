# Professional Event Anchor Portfolio

Production-ready MERN stack portfolio for a professional event anchor, including:

- animated public portfolio
- admin authentication and management
- events, gallery, feedback, and inquiry workflows
- Vercel-ready frontend and Render-ready backend configuration

## Run locally

1. Copy `.env.example` to `.env` in the project root.
2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```

Frontend runs at `http://localhost:5173` and backend runs at `http://localhost:5000`.

## Production

- Frontend: deploy `client` to Vercel
- Backend: deploy `server` to Render
- Database: MongoDB Atlas

Use the deployment notes inside the app configs and environment examples.
